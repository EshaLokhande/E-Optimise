# E OPTIMISE

A VS Code extension that helps developers visualize, analyze and optimize their code.

## Features

- Visualise functions as diagrams
- Get Big-O complexity analysis
- Optimise your code with AI

## Project Structure

- `backend/` - Express.js API server
- `extension/` - VS Code Extension
- `dashboard/` - Web Dashboard
